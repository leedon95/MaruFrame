 -- 소개글 (개발자의 헛소리) --


 안녕하세요! 마루 프레임워크 개발자입니다. 사실 말만 프레임워크지 라이브러리에 가깝습니다.

 마루웨어 프레임워크는 누구나 쉽게 이미지슬라이더나 패럴렉스 스크롤링을 사용하기 위해 고안되었습니다.

 제작은 책에서 나온 에제를 수정하여 만들거나 개발자의 머릿속에서 떠오르는대로 만들어집니다. // 누가 물어봤어;

 사실 제 입맛대로 만든게 대부분이라 마음에 들지 않으시는 분들도 계실텐데요. // 안쓰시면 됩니다.... :(
  CSS 파일안에 태그에 주석을 하나하나 다 붙임여놓았습니다. 코드를 조금씩 수정하여 입맛에 맞게 사용하시면 되겠습니다.

 마지막으로 프레임워크가 적용이 된 예제파일을 하나씩 (대충) 만들어두었으니 관람하시고 하나씩 빼서 쓰시면 됩니다.



 -- 개발자가 생각하는 마루웨어의 장점 (이 있을까) --

 1. 일단 첫번째로 한글을 지원합니다. 만든사람이 한국사람이기 때문입니다.

 2. 복붙만 하면 쉽게 만들 수 있습니다. 사실 제가 그렇게 사이트를 만들고 있습니다.

 3. Css 파일에 주석을 하나씩 달아두었습니다. 덕분에 쉽게 Css 파일 수정이 가능합니다.

 4. 오픈소스입니다. // 이게 장점이냐;

 5. 현재 패럴렉스 스크롤링과 이미지 갤러리를 제외한 나머지는 모두 반응형으로 만들어져 있습니다.
     실제로 브라우저 너비 조정을 하시면 애들이 알아서 줄어들었다가 커졌다가 합니다. 포트폴리오 만들때 용이하게 씁시다.


 -- 본문 (API) --

 0. 마루웨어 프레임워크 불러오기

 링크 태그를 이용하여 불러오시면 되겠습니다.

 소스 : <link href="MaruFrame.css" rel="stylesheet">



 1. 버튼

 마루웨어 프레임워크는 여러가지 유형의 버튼 디자인을 제공하고 있습니다.
 
 간단한 애니메이션을 이용한 버튼들과 고스트버튼을 지원합니다.

 사이즈별로 기본값, Middle(중간크기), Big(큰크기), Horizon (너비 100%)가 있으며, 상황에 따라 불러와 쓰시면 됩니다.

 html 소스 :: <input type="button" class="BasicButton" value="BasicButton입니다."> //기본 버튼
	      <a href="#" class="Ghost-button">Ghost-button입니다.</a> //고스트버튼

 나머지 버튼들은 button.html에서 자세하게 확인이 가능합니다.



 2. 이미지 갤러리

 
 이미지 갤러리를 사용하기 위해선 Galleybody를 호출해야 합니다. ( 고정값을 사용하기 때문에 안하면 이상하게 나옵니다.)

 <div class="GalleryBody"> <!-- 바디 -->

   <div class="IMGgallery3"> <!-- Div로 묶어줌. IMGgallery3은 화면에 3개출력 IMGgallery2는 화면에 2개출력 -->
     <a target="_parent" href="#이미지를 넣어주세요!#"> <!-- 클릭시 이미지 출력 -->
       <img src="#이미지를 넣어주세요!#"> <!-- 화면에 보이는 이미지 -->
     </a>
    <div class="desc">MARUWARE</div> <!-- 이미지 하단에 글씨 출력 -->
   </div> <!-- IMGgallery3 끝 -->

 </div> <!-- GalleryBody끝 -->

 위 소스 복붙하세요. 그럼 잘 만들어집니다. 한개의 행에 IMGgallery3은 3개가 들어갈 수 있고 2는 2개가 들어갑니다.
  a 태그안에 target은 여러가지 속성이 들어갈 수 있습니다. 구글에 치면 어떤 어떤 속성이 있는지 다 자세하게 나오니까
  구글링 하시면 알아서 다나옵니다. 걔중에 보고 마음에 드는거 박으시면 됩니다. 저도 그렇게 하고있어요.

 IMGgallery 1,4,5,6은 추후 추가예정입니다. 현재 자격증 공부를 하고 있어서 시간이 많이 없습니다. // 변명하지마


 3. 텍스트박스

 Article 태그를 이용한 텍스트박스도 구현되어 있습니다. 전부 반응형으로 만들어져 알아서 커졌다 작아졌다 합니다.

     <div class="GalleryBody">

      <article class="articlemd2"> <!--  articlemd2는 한 행에 2개의 아티클박스를 넣을 수 있습니다. 3은 한 행에 3개 -->
        <p class="articlefont1">이미지 게시판</p> <!-- 제목이라고 보시면 됩니다. 반응형 폰트로 알아서 크기조절 됩니다. -->
        <p class="articlefont2"> <!-- 내용이라고 보시면 됩니다. 반응형 폰트.. -->
          &nbsp;마루웨어 프레임워크는 간단한 이미지 게시판을 제공합니다.<br>
          &nbsp;기본적인 틀 제공 뿐만아니라 이미지 쿼리를 이용한 반응형으로 제작되어
          태블릿이나 모바일 환경에서도 관람이 가능합니다.<br>
          &nbsp;사용하기 쉬운 태그로 누구나 쉽게 코딩이 가능하도록 제작되으며 이미지의 갯수에 제한이 없습니다.
        </p>
      </article>  <!-- 종료 -->

 아직 hover 속성을 이용한 추가 애니메이션은 없지만 1.3에서 추가 될 예정입니다.
 Article태그는 텍스트환경을 염두에 두고 소스를 작성하였기 때문에 이미지 보다는 글자를 넣으시는것이 현명합니다.


 4. 패럴랙스 스크롤링 

 이것 또한 복붙만 잘하면 끝납니다.

<div class="pimg" style="background-image:url('#이미지를 넣어주세요!#');"> <!-- style안에 이미지 넣으면 끝나요 -->
 <div class="intext-center" style="top:50%"> <!-- intext-center는 텍스트 가운데 -->
  <font size="14" style="color:black;">마루 프레임워크에 오신걸 환영합니다.</font> <!-- 글을 넣는 곳입니다. 글을 넣지 않아도 무관합니다. -->
 </div>
</div>

 패럴렉스 스크롤링은 좀 복잡해보이지만 정말 단순하게 작동합니다. pimg 클래스는 큰 백그라운드 이미지가 위치하는 곳으로서,
제일 위에 선언해주고 style 태그에 background-image:url을 선언해줍니다. 그리고 #이미지를 넣어주세요!# << 이부분에
이미지를 삽입해주시면 되겠습니다.
 다음으로 div class="intext"를 넣어야 하는데요.

1. intext-center는 글자가 중앙에 위치하고 
2. intext-left는 글자가 왼쪽에 위치하고 
3. intext-right는 글자가 오른쪽에 위치합니다.

뒤에 style을 넣은것은 일부로 넣은건데요. 굳이 화면 정 중앙에 넣고싶지 않은 분들을 위해 일부로 css파일에 소스를 넣지 않았습니다.
화면 중앙에 박고있을때 top:50%을 넣어주시면 되겠습니다.



5. 리모컨 기능

너무 쉽습니다. 복붙하세요 알아서 다 만들어집니다.

<div id="Remocon"> <!-- 리모컨을 열어주는 코드입니다. -->
	<center>
		<h2 id="RemoconTop">Remocon</h2> <!-- 리모컨의 머리부분입니다. -->
		<input type='button' value='△' style="margin-bottom:15px; width:30px; height:30px;" onclick="divtop()">
     <!--저는 버튼을 넣었습니다. 리모컨 내부에 여러분들이 넣고싶은걸 넣으시면 됩니다. -->
		<input type='button' value='▽' style="margin-bottom:15px; width:30px; height:30px;" onclick="divbot()">
	</center>
</div>

디자인이 마음에 안드시는 분들은 css파일을 열어서 /*리모컨입니다.*/을 검색하세요.
그럼 Remecon이 나오는데 거기서 주석보고 수정하시면 됩니다.


6. 이미지 슬라이더

이미지 슬라이더는 좀 큽니다. 하지만 우리에겐 복붙이 있습니다.

 <div class="slideshow-container"> <!-- 슬라이더를 열어줍니다. -->

  <div class="mySlides fadeIMG"> <!-- 이미지가 위치하는 곳입니다. -->
    <img src="#이미지를 넣어주세요." style="width:100%"> <!-- 이미지를 넣는곳입니다. 경로에 맞게 설정해주세요. -->
    <div class="Texton-IMG">이미지 슬라이더1</div> <!-- 글을 넣으세요. 이미지만 보이고 싶으면 안넣어도 됩니다.-->
  </div>

  <div class="mySlides fadeIMG">
    <img src="#이미지를 넣어주세요." style="width:100%">
    <div class="Texton-IMG">이미지 슬라이더1</div> 
  </div>

</div>

<div style="text-align:center">
  <span class="dotonIMG" onclick="currentSlide(1)"></span> <!-- 이미지 슬라이더 밑에 O를 만들어줍니다.-->
  <span class="dotonIMG" onclick="currentSlide(2)"></span> <!-- 한개씩 늘어날때마다 (1),(2),(3)순으로 만들어 주셔야 합니다. -->
</div>


<script type="text/javascript" src="MaruFrame.js"></script> <!-- 꼭 쓰세요 안하면 안 됌. -->

위 소스를 붙여다 쓰세요 알아서 될겁니다.

 이미지 슬라이더는 마루프레임워크에서 난이도가 가장 높습니다. 그 이유는 링크해야 하는것이 1개 더 있기 떄문이지요. // 별로 안어려운데
먼저 이미지 슬라이더를 열고 작성해줍니다. 옆에 주석 있으니 보기 쉬울겁니다. 이미지 한개를 더 늘리려면 mySlides fadeIMG를 열심히 복붙하세요.
하지만 한개씩 늘어날때마다 dotonIMG를 한개씩 더 늘려주셔야 합니다. 온클릭 태그 안에 (1),(2),(3)이렇게 한개씩 늘려가시면 됩니다.
제가 실험해봤는데 (10)까지도 거뜬하게 됩니다.
 그리고 마지막으로 밑에 스크립트 호출을 하나 해야합니다. 저거 안하면 에러납니다. 귀찮아도 꼭 해주세요.
